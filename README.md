# FitCheck
A wardrobe managment iOS app built for COMP 499 Capstone. 

## Structure
-'ios/' - Swift/Xcode iOS app (TormentNexus)
- 'backend/' -Node.js/ Express REST API

## Running the Backend
'''bash
cd backend
node src/index.js
'''

## Tech Stack
Swift, Node.js, PostfreSQL, AWS S3, JWT, OpenWeatherMap API
